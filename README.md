**DistroUpdater**

A simple python script to update most Linux distros in one simple line.

** How to install (one liner) **
```sh
curl -LO https://github.com/digitalencryption/DistroUpdater/distroupdaterinstall.sh && bash distroupdaterinstall.sh
```


Usage 
```sh
python3 DistroUpdater.py 
```
