**DistroUpdater**

A simple python script to update most Linux distros in one simple line.

**How to install (one liner)**
```sh
curl -LO https://raw.githubusercontent.com/DigitalEncryption/DistroUpdater/refs/heads/main/distroupdaterinstall.sh && bash distroupdaterinstall.sh
```


**Usage** 
```sh
distroupdater
```
